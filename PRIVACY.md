# Privacy Policy for Eternal Project

*Last updated: March 2026*

We value your privacy. This project is an open-source initiative and operates under the following principles:

### 1. Data Collection
This project does *not* collect any personal data from its users. Any data processed stays within the user's local environment.

### 2. Cookies
We do not use tracking cookies or any third-party analytics that monitor your behavior.

### 3. Transparency
All code is open-source. You can audit the repository to ensure no hidden data collection mechanisms exist.

### 4. Contact
If you have questions regarding this policy, you can open an "Issue" in this repository.
